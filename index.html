Bienvenido al Portal de Transparencia

Este portal proporciona acceso público a documentos oficiales, informes financieros y datos de transparencia de la Municipalidad de Carmen de Areco. Todos los documentos han sido recolectados, organizados y categorizados para facilitar el acceso a la información pública.

El portal contiene información financiera desde 2017 hasta 2025, incluyendo:

  • Ejecución presupuestaria y estados financieros
  • Informes de ejecución de gastos y recursos
  • Documentos de personal y recursos humanos
  • Contrataciones públicas y licitaciones
  • Declaraciones patrimoniales
  • Informes de salud pública y programas sociales

Todos los datos están disponibles en formatos abiertos y pueden ser descargados para análisis independiente.

Datos Abiertos

Todos los documentos y datos están disponibles en formatos abiertos para descarga y análisis independiente:

Recurso Formato Descripción Enlace
Inventario Completo de Documentos CSV, JSON Listado completo de todos los documentos organizados Descargar
Documentos en Texto Plano Markdown Versiones en texto de todos los documentos PDF Explorar
Datos Estructurados JSON Datos extraídos en formato estructurado Acceder
Metadatos de Documentos JSON Información sobre cada documento (fecha, tipo, tamaño) Ver

API de Datos

Los datos también están disponibles a través de una API REST:

// Ejemplo de acceso a documentos por año
GET /api/documents/2023
GET /api/documents/category/execucion-gastos
GET /api/documents/search?q=presupuesto
